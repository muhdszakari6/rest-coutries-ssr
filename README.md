# Fun with countries
Angular PWA application with SSR and SSG showing list of countries in the world. 
Technologies used: angular, angular universal, angular pwa, ngrx, jasmine, cypresss.

##### Install npm modules

```shell
npm install
```

##### Run application

```shell
npm start
```

##### Run unit tests 

```shell
ng test
```

##### Run e2e tests 

```shell
npm run cypress:run 
```

##### Build

```shell
npm run build
```
##### Build SSR

```shell
npm run build:ssr
```

##### Run SSR

```shell
npm run serve:ssr
```

##### Prerender

```shell
npm run prerender
```
